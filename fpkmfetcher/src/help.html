FPKM files downloader

This program is created in order to download FPKM (Fragments Per Kilobase of transcript per Million mapped reads) files from the NCI's Genomic Data Commons (GDC). More information about GDC and FPKM files you could fined on https://gdc.cancer.gov/

The program is using The GDC Application Programming Interface (API) to connect to the server and download all necessary files

Guide

In order to download FPKM files you should choose or fill next points:

1) Choose cancer type (Currently only Breast cancer is available)\n

2) Write number of files that have to be downloaded of each stage

3) Choose which stages of cancer have to be downloaded

4) Choose or fill manualy the directory, where files should be downloaded

5) Fill checkbox if files have to be joined and choose type of joining: if there are more than 200 files per stage is preferred to choose append method, but it will take much longer

6) Trigger start button
